# email-lint

Lint email HTML for client compatibility using [caniemail](https://www.caniemail.com/) data.

| Package | Description |
|---|---|
| [`@email-lint/core`](./packages/email-lint) | Core linter + CLI |
| [`@email-lint/react-email`](./packages/react-email) | React Email integration |

## Development

```sh
pnpm install
pnpm build
pnpm test
```
